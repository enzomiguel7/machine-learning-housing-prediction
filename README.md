# Machine Learning Housing Price Prediction

This project predicts house prices in California using machine learning. 
It covers the full ML pipeline: data exploration, preprocessing, feature engineering, 
and model evaluation with various algorithms including Linear Regression and Random Forest.

## Features
- Exploratory Data Analysis (EDA) with visualizations
- Handling missing values and categorical data
- Train/Test split and model evaluation
- Hyperparameter tuning and performance metrics

## Libraries
- Python 3.x
- pandas, numpy, matplotlib, seaborn
- scikit-learn

## Results
- Achieved RMSE of [valor] on the test set
- Comparison of multiple models for best performance

## How to run
Clone the repository, install dependencies from `requirements.txt`, and run the notebooks in order.
